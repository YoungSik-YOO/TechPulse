# SCR-009: 그룹 상세 화면

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-009 |
| 경로 | `/groups/:groupId` |
| 설명 | 그룹 정보, 멤버 목록, 그룹 전용 게시물 피드 |

## 레이아웃

```
┌─────────────────────────────────┐
│  ← 뒤로                         │
├─────────────────────────────────┤
│  🖼️ [그룹 커버 이미지]           │
│                                 │
│  AI 연구 모임                    │
│  "최신 AI 논문과 기술을 공유"     │
│  👥 45명  📝 128개 게시물        │
│  [AI] [LLM] [딥러닝]            │
│                                 │
│  [가입하기 / 탈퇴하기]           │
├─────────────────────────────────┤
│  [게시물] [멤버(45)]              │  ← 탭
├─────────────────────────────────┤
│  ┌─────────────────────────┐    │
│  │ 👤 홍길동  · 10분 전     │    │
│  │ 새로운 LLM 파인튜닝...   │    │
│  │ 💬 2  ❤️ 8              │    │
│  └─────────────────────────┘    │
│  ...무한 스크롤...               │
│                                 │
│              [＋]                │  ← FAB(그룹 게시물 작성)
└─────────────────────────────────┘
```

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 뒤로가기 | `btn-back` | 아이콘 버튼 | SCR-008 이동 |
| 그룹 커버 | `img-group-cover` | 이미지 | 표시 전용 |
| 가입/탈퇴 버튼 | `btn-join-leave` | 토글 버튼 | Firestore 멤버 추가/삭제 |
| 탭 (게시물/멤버) | `tab-group-content` | 탭 | 게시물 피드 ↔ 멤버 목록 전환 |
| 그룹 게시물 카드 | `card-group-post-{id}` | 카드 | 클릭 시 SCR-007 이동 |
| 멤버 리스트 | `list-members` | 리스트 | 멤버 프로필 사진, 닉네임, 역할 표시 |
| FAB 버튼 | `fab-create-group-post` | FAB | 그룹 게시물 작성 모달 오픈 (멤버만) |
