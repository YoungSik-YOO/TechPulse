# SCR-003: 프로필 설정 화면

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-003 |
| 경로 | `/profile/edit` |
| 설명 | 프로필 사진, 닉네임, 관심분야, 기술스택 등을 설정 |

```

├─────────────────────────────────┤

│         └──────┘                │
│                                 │
│  닉네임:  ┌───────────────┐     │
│           │               │     │

│           │               │     │
│           │               │     │
│           └───────────────┘     │
```

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 프로필 이미지 | `upload-profile-img` | 이미지 업로더 | Firebase Storage 업로드 |
| 닉네임 | `input-displayname` | 텍스트 입력 | 중복 확인 (Firestore 쿼리) |
| 자기소개 | `input-bio` | 텍스트에어리어 | 최대 200자 |
| 관심 분야 | `tag-interests` | 태그 선택 (멀티) | 사전 정의된 태그 목록에서 선택 |
| 기술 스택 | `tag-techstack` | 태그 입력 | 자유 입력, 추가/삭제 가능 |
| 경력 | `input-career` | 텍스트 입력 | 자유 입력 |
| GitHub 링크 | `input-github` | URL 입력 | URL 형식 검증 |
| 블로그 링크 | `input-blog` | URL 입력 | URL 형식 검증 |
| 지역 | `select-region` | 드롭다운 | 시/도 목록에서 선택 |
| 저장 버튼 | `btn-save-profile` | 버튼 | Firestore 프로필 업데이트 → SCR-005 이동 |
