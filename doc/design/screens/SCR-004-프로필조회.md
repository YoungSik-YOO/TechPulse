# SCR-004: 프로필 조회 화면

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-004 |
| 경로 | `/profile/:userId` |
| 설명 | 사용자의 프로필 정보와 작성한 게시물 목록 표시 |

```
│  └──────┘  @hongildong           │

```

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 뒤로가기 | `btn-back` | 아이콘 버튼 | 이전 화면으로 이동 |
| 프로필 이미지 | `img-profile` | 이미지 | 표시 전용 |
| 닉네임 | `text-displayname` | 텍스트 | 표시 전용 |
| 관심분야/기술스택 | `tags-profile` | 태그 칩 목록 | 클릭 시 해당 태그 검색 |
| 외부 링크 | `link-external` | 링크 | 새 탭에서 열기 |
| 프로필 편집 버튼 | `btn-edit-profile` | 버튼 | SCR-003 이동 (본인만 표시) |
| 게시물 목록 | `list-user-posts` | 무한 스크롤 리스트 | PostCard 컴포넌트 사용, 클릭 시 SCR-007 이동 |
