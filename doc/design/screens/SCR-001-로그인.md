# SCR-001: 로그인 화면

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-001 |
| 경로 | `/login` |
| 설명 | 이메일/비밀번호 또는 소셜 계정으로 로그인 |

## 레이아웃

```
┌─────────────────────────────────┐
│          TechPulse 로고          │
│     "AI/IT 지식 공유 커뮤니티"    │
├─────────────────────────────────┤
│  ┌─────────────────────────┐    │
│  │  📧 이메일 입력          │    │
│  └─────────────────────────┘    │
│  ┌─────────────────────────┐    │
│  │  🔒 비밀번호 입력        │    │
│  └─────────────────────────┘    │
│                                 │
│  [      로그인 버튼       ]      │
│                                 │
│  ──────── 또는 ────────         │
│                                 │
│  [  🔵 Google로 로그인    ]      │
│  [  ⚫ GitHub로 로그인    ]      │
│                                 │
│  계정이 없으신가요? [회원가입]    │
└─────────────────────────────────┘
```

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 이메일 입력 | `input-email` | 텍스트 입력 | 이메일 형식 검증 |
| 비밀번호 입력 | `input-password` | 비밀번호 입력 | 최소 8자 |
| 로그인 버튼 | `btn-login` | 버튼 | Firebase Auth 이메일 로그인 → SCR-005 이동 |
| Google 로그인 | `btn-google` | 버튼 | Firebase Auth Google 소셜 로그인 |
| GitHub 로그인 | `btn-github` | 버튼 | Firebase Auth GitHub 소셜 로그인 |
| 회원가입 링크 | `link-signup` | 링크 | SCR-002 이동 |
