# SCR-002: 회원가입 화면

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-002 |
| 경로 | `/signup` |
| 설명 | 이메일/비밀번호 또는 소셜 계정으로 회원가입 |

## 레이아웃

```
┌─────────────────────────────────┐
│     ← 뒤로    회원가입           │
├─────────────────────────────────┤
│  ┌─────────────────────────┐    │
│  │  📧 이메일 입력          │    │
│  └─────────────────────────┘    │
│  ┌─────────────────────────┐    │
│  │  🔒 비밀번호 입력        │    │
│  └─────────────────────────┘    │
│  ┌─────────────────────────┐    │
│  │  🔒 비밀번호 확인        │    │
│  └─────────────────────────┘    │
│                                 │
│  [     회원가입 버튼      ]      │
│                                 │
│  ──────── 또는 ────────         │
│                                 │
│  [  🔵 Google로 가입      ]      │
│  [  ⚫ GitHub로 가입      ]      │
│                                 │
│  이미 계정이 있나요? [로그인]    │
└─────────────────────────────────┘
```

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 이메일 입력 | `input-signup-email` | 텍스트 입력 | 이메일 형식 검증 |
| 비밀번호 입력 | `input-signup-pw` | 비밀번호 입력 | 최소 8자, 특수문자 포함 |
| 비밀번호 확인 | `input-signup-pw-confirm` | 비밀번호 입력 | 비밀번호 일치 확인 |
| 회원가입 버튼 | `btn-signup` | 버튼 | Firebase Auth 계정 생성 → SCR-003 이동 |
| 소셜 가입 | `btn-google-signup`, `btn-github-signup` | 버튼 | 소셜 인증 → SCR-003 이동 |
| 로그인 링크 | `link-login` | 링크 | SCR-001 이동 |
