# SCR-005: 메인 피드 화면

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-005 |
| 경로 | `/` (홈) |
| 설명 | 모든 공개 게시물이 최신순으로 표시되는 메인 피드 |

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 검색 아이콘 | `btn-search` | 아이콘 버튼 | SCR-011 이동 (Phase 2+) |
| 알림 아이콘 | `btn-notification` | 아이콘 버튼 | SCR-014 이동 (Phase 3+) |
