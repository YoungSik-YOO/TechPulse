# SCR-008: 그룹 탐색 화면

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-008 |
| 경로 | `/groups` |
| 설명 | 공개 그룹 목록 탐색, 카테고리/지역 필터링 |

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 필터 드롭다운 | `filter-category`, `filter-region` | 드롭다운 | Firestore 쿼리 필터 적용 |
| 그룹 만들기 버튼 | `btn-create-group` | 버튼 | SCR-010 이동 |
| 그룹 카드 | `card-group-{id}` | 카드 | 클릭 시 SCR-009 이동 |
| 가입하기 버튼 | `btn-join-{groupId}` | 버튼 | Firestore 멤버 추가 → 버튼 '가입됨'으로 변경 |
