# SCR-010: 그룹 생성 화면

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-010 |
| 경로 | `/groups/create` |
| 설명 | 새로운 그룹을 생성하는 폼 |

│         └──────┘                │
│                                 │
│  그룹명:  ┌───────────────┐     │
│           │               │     │

│  설명:    ┌───────────────┐     │
│           │               │     │
│           │               │     │
│           └───────────────┘     │
│                                 │
│  카테고리: ○ 관심 분야  ○ 지역   │

│  공개 설정: ○ 공개  ○ 비공개     │

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 그룹 이미지 | `upload-group-img` | 이미지 업로더 | Storage 업로드 |
| 그룹명 | `input-group-name` | 텍스트 입력 | 필수, 중복 확인 |
| 설명 | `input-group-desc` | 텍스트에어리어 | 최대 500자 |
| 카테고리 | `radio-category` | 라디오 버튼 | interest / region |
| 태그 | `tag-group-tags` | 태그 입력 | 자유 입력 |
| 공개 설정 | `radio-visibility` | 라디오 버튼 | public / private |
| 만들기 버튼 | `btn-create-group` | 버튼 | Firestore 그룹 문서 생성 → SCR-009 이동 |
