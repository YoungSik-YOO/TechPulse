# SCR-006: 게시물 작성 모달

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-006 |
| 경로 | 모달 (오버레이) |
| 설명 | 마크다운 텍스트, 코드, 이미지, 해시태그를 입력하여 게시물 작성 |

```
┌─────────────────────────────────┐
│  ✕ 닫기        게시물 작성       │

│  👤 홍길동                       │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  │  무엇을 공유하시겠어요?   │    │
│  │  (마크다운 지원)          │    │
│  │                         │    │
│  │                         │    │
│  └─────────────────────────┘    │
│                                 │
│  ┌─────┐ ┌─────┐ ┌─────┐       │
│  │ 📷 │ │ 🎬 │ │ </> │       │
│  │이미지│ │동영상│ │코드 │       │
│  └─────┘ └─────┘ └─────┘       │
│                                 │
│  해시태그: [#AI ×] [#Python ×]   │
│            [+ 태그 추가]         │

```

## UI 요소

| 요소 | ID | 유형 | 동작 |
|------|----|------|------|
| 닫기 | `btn-close-modal` | 아이콘 버튼 | 모달 닫기 |
| 콘텐츠 입력 | `textarea-content` | 텍스트에어리어 | 마크다운 입력 |
| 이미지 첨부 | `btn-attach-image` | 버튼 | 파일 선택 → Storage 업로드 |
| 동영상 첨부 | `btn-attach-video` | 버튼 | 파일 선택 → Storage 업로드 |
| 코드 블록 | `btn-insert-code` | 버튼 | 코드 블록 ``` 삽입 + 언어 선택 |
| 해시태그 | `input-hashtag` | 태그 입력 | # 입력 시 자동완성 |
| 미리보기 토글 | `toggle-preview` | 토글 | 마크다운 렌더링 미리보기 |
| 게시 버튼 | `btn-publish` | 버튼 | Firestore에 게시물 저장 → 모달 닫기 |
