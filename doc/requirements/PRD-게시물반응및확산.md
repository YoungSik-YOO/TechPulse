# PRD: 게시물 반응 및 확산

## 기능 개요
사용자가 다른 사용자의 게시물에 댓글, 좋아요, 리포스트, 북마크 등의 반응을 통해 상호작용하고, 유용한 콘텐츠를 확산시키는 기능입니다.

## 우선순위: 🔴 필수

## 상세 기능 명세

### F4-1. 댓글
| 항목 | 내용 |
|------|------|
| 기능명 | 댓글 |
| 설명 | 사용자는 게시물에 텍스트 댓글을 작성할 수 있으며, 댓글에 대한 답글(대댓글)도 가능하다. |
| 입력 | 댓글 텍스트 |
| 처리 | Firestore에 댓글 서브컬렉션으로 저장, 게시물 작성자에게 알림 발송 |
| 출력 | 게시물 하단에 댓글 목록 표시 |
| 예외 | 빈 댓글 작성 불가 |

### F4-2. 좋아요
| 항목 | 내용 |
|------|------|
| 기능명 | 좋아요 |
| 설명 | 사용자는 게시물에 좋아요를 누르거나 취소할 수 있다. |
| 입력 | 좋아요 버튼 클릭 (토글) |
| 처리 | Firestore에 좋아요 관계 문서 생성/삭제, 카운트 갱신 |
| 출력 | 좋아요 아이콘 상태 변경, 좋아요 수 표시 갱신 |
| 예외 | 중복 좋아요 방지 (토글 방식) |

### F4-3. 리포스트 (리트윗)
| 항목 | 내용 |
|------|------|
| 기능명 | 리포스트 |
| 설명 | 사용자는 다른 사용자의 게시물을 자신의 팔로워에게 리포스트(공유)할 수 있다. 선택적으로 인용 코멘트를 추가할 수 있다. |
| 입력 | 리포스트 버튼 클릭, (선택) 인용 코멘트 |
| 처리 | 원본 게시물 참조와 함께 리포스트 문서를 Firestore에 생성 |
| 출력 | 자신의 프로필 및 팔로워 피드에 리포스트 표시 |
| 예외 | 이미 리포스트한 게시물 → 리포스트 취소 가능 |

### F4-4. 북마크
| 항목 | 내용 |
|------|------|
| 기능명 | 북마크 (저장) |
| 설명 | 사용자는 유용한 게시물을 북마크에 저장하여 나중에 쉽게 찾아볼 수 있다. |
| 입력 | 북마크 버튼 클릭 (토글) |
| 처리 | Firestore 사용자 북마크 컬렉션에 게시물 참조 저장/삭제 |
| 출력 | 북마크 아이콘 상태 변경, 프로필의 '북마크' 탭에서 저장된 게시물 조회 가능 |
| 예외 | 삭제된 게시물은 북마크 목록에서 '삭제된 게시물' 표시 |

## 화면 구성

| 화면 | 주요 요소 |
|------|-----------|
| 게시물 하단 액션 바 | 댓글 아이콘(수), 좋아요 아이콘(수), 리포스트 아이콘(수), 북마크 아이콘 |
| 댓글 영역 | 댓글 입력 폼, 댓글 목록 (프로필 사진, 닉네임, 내용, 시간) |
| 북마크 목록 | 프로필 > 북마크 탭에서 저장된 게시물 리스트 |
