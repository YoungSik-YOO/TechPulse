# PRD: 탐색 및 트렌딩

## 기능 개요
사용자가 키워드·해시태그로 기술 콘텐츠를 검색하고, AI/IT 분야의 실시간 인기 주제를 확인하며, 관심 있을 만한 사용자나 그룹을 발견할 수 있는 기능입니다.

## 우선순위: 🟡 중요

## 상세 기능 명세

### F8-1. 검색
| 항목 | 내용 |
|------|------|
| 기능명 | 키워드/해시태그 검색 |
| 설명 | 사용자는 키워드나 해시태그를 통해 게시물, 사용자, 그룹을 검색할 수 있다. |
| 입력 | 검색 키워드 텍스트 |
| 처리 | Firestore 쿼리 (텍스트 인덱싱 또는 외부 검색 엔진 연동), 결과를 게시물/사용자/그룹 탭으로 분류 |
| 출력 | 검색 결과 목록 (게시물, 사용자, 그룹 탭 분류) |
| 예외 | 결과 없음 → '검색 결과가 없습니다' 메시지 + 추천 키워드 제안 |

### F8-2. 트렌딩 토픽
| 항목 | 내용 |
|------|------|
| 기능명 | 트렌딩 토픽 |
| 설명 | AI/IT 분야에서 최근 일정 시간 동안 가장 많이 언급된 해시태그나 주제를 실시간으로 표시한다. |
| 입력 | 자동 갱신 (주기적 집계) |
| 처리 | 최근 24시간 이내 게시물에서 해시태그 빈도를 집계하여 순위 산출 |
| 출력 | 트렌딩 랭킹 리스트 (순위, 해시태그, 게시물 수) |
| 예외 | 데이터 부족 시 → 기본 추천 주제 표시 |

### F8-3. 추천 사용자/그룹
| 항목 | 내용 |
|------|------|
| 기능명 | 추천 사용자/그룹 |
| 설명 | 사용자의 관심 분야, 팔로우 관계, 활동 기록을 바탕으로 관심 있을 만한 사용자와 그룹을 추천한다. |
| 입력 | 자동 (사용자 프로필 데이터 기반) |
| 처리 | 관심 분야 태그 매칭, 팔로잉의 팔로잉(2단계 관계) 기반 추천 알고리즘 |
| 출력 | 추천 사용자 카드 (프로필 사진, 닉네임, 관심 분야, 팔로우 버튼) |
| 예외 | 프로필 미설정 시 → 인기 사용자/그룹 기본 추천 |

## 화면 구성

| 화면 | 주요 요소 |
|------|-----------|
| 탐색/검색 페이지 | 검색 입력 바, 탭 (게시물/사용자/그룹), 트렌딩 섹션, 추천 사용자 리스트 |
| 트렌딩 섹션 | 순위 번호, 해시태그, 관련 게시물 수, 클릭 시 해당 해시태그 검색 결과 이동 |
