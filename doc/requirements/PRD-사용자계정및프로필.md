# PRD: 사용자 계정 및 프로필

## 기능 개요
사용자가 서비스에 가입하고 로그인하며, 자신만의 기술 프로필을 설정하여 전문성을 표현할 수 있는 기능입니다.

## 우선순위: 🔴 필수

## 상세 기능 명세

### F1-1. 회원가입
| 항목 | 내용 |
|------|------|
| 기능명 | 회원가입 |
| 설명 | 사용자는 이메일, Google 계정 또는 GitHub 계정을 통해 회원가입을 할 수 있다. |
| 입력 | 이메일/비밀번호 또는 소셜 계정 인증 |
| 처리 | Firebase Authentication을 통한 사용자 생성 |
| 출력 | 가입 완료 후 프로필 설정 화면으로 이동 |
| 예외 | 이미 가입된 이메일 → 에러 메시지 표시, 비밀번호 규칙 미충족 → 안내 메시지 |

### F1-2. 로그인/로그아웃
| 항목 | 내용 |
|------|------|
| 기능명 | 로그인/로그아웃 |
| 설명 | 사용자는 가입한 계정으로 로그인/로그아웃을 할 수 있다. |
| 입력 | 이메일/비밀번호 또는 소셜 로그인 버튼 클릭 |
| 처리 | Firebase Authentication 인증 → JWT 토큰 발급 |
| 출력 | 로그인 성공 시 메인 피드로 이동, 로그아웃 시 로그인 화면으로 이동 |
| 예외 | 잘못된 비밀번호 → 에러 메시지, 계정 잠김 → 안내 메시지 |

### F1-3. 프로필 설정
| 항목 | 내용 |
|------|------|
| 기능명 | 프로필 설정 |
| 설명 | 사용자는 프로필 사진, 자기소개, 관심 분야, 기술 스택, 경력, GitHub/블로그 링크를 설정할 수 있다. |
| 입력 | 프로필 이미지, 닉네임, 자기소개 텍스트, 관심 분야 태그, 기술 스택 태그, 외부 링크 |
| 처리 | Firestore에 사용자 프로필 문서 업데이트 |
| 출력 | 업데이트된 프로필 페이지 표시 |
| 예외 | 닉네임 중복 → 에러 메시지, 이미지 파일 사이즈 초과 → 안내 메시지 |

## 화면 구성

| 화면 | 주요 요소 |
|------|-----------|
| 회원가입 화면 | 이메일/비밀번호 입력 폼, Google/GitHub 소셜 로그인 버튼 |
| 로그인 화면 | 이메일/비밀번호 입력 폼, 소셜 로그인 버튼, 비밀번호 찾기 링크 |
| 프로필 설정 화면 | 프로필 이미지 업로드, 닉네임, 자기소개, 관심 분야 태그 선택, 기술 스택 입력, 외부 링크 입력 |
| 프로필 조회 화면 | 사용자 정보 표시, 게시물 목록, 팔로워/팔로잉 수 |
