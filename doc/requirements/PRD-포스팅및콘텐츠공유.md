# PRD: 포스팅 및 콘텐츠 공유

## 기능 개요
사용자가 코드, 이미지, 동영상, 마크다운 등 다양한 형식의 기술 콘텐츠를 작성하고 공유할 수 있는 핵심 기능입니다.

## 우선순위: 🔴 필수

## 상세 기능 명세

### F3-1. 게시물 작성
| 항목 | 내용 |
|------|------|
| 기능명 | 게시물 작성 |
| 설명 | 사용자는 텍스트를 입력하고, 코드 블록·이미지·동영상·해시태그를 첨부하여 게시물을 작성할 수 있다. |
| 입력 | 텍스트 내용, 첨부 미디어 (이미지/동영상), 해시태그 |
| 처리 | 미디어 파일 → Firebase Storage 업로드, 게시물 데이터 → Firestore 저장 |
| 출력 | 작성된 게시물이 피드에 표시 |
| 예외 | 텍스트 없이 미디어만 게시 가능, 파일 크기 제한 초과 시 안내 메시지 |

### F3-2. 마크다운 및 코드 하이라이팅
| 항목 | 내용 |
|------|------|
| 기능명 | 마크다운 / 코드 하이라이팅 |
| 설명 | 게시물 작성 시 마크다운 문법을 지원하며, 코드 블록에는 언어별 구문 강조(syntax highlighting)가 자동 적용된다. |
| 입력 | 마크다운 형식의 텍스트, ` ``` ` 코드 블록 |
| 처리 | 마크다운 파서를 통해 렌더링, 코드 하이라이팅 라이브러리(예: Prism.js, highlight.js) 적용 |
| 출력 | 포맷이 적용된 게시물 렌더링 |
| 예외 | 지원하지 않는 마크다운 문법 → 원본 텍스트로 표시 |

### F3-3. 링크 프리뷰
| 항목 | 내용 |
|------|------|
| 기능명 | 링크 프리뷰 (OG 태그) |
| 설명 | 게시물에 URL을 포함하면 해당 링크의 제목·설명·썸네일 이미지가 카드 형태로 자동 표시된다. |
| 입력 | 게시물 내 URL |
| 처리 | 서버에서 해당 URL의 OG 메타 태그를 파싱하여 프리뷰 데이터 생성 |
| 출력 | 링크 프리뷰 카드 (제목, 설명, 이미지, 도메인명) |
| 예외 | OG 태그 미지원 사이트 → URL 텍스트만 표시 |

### F3-4. 동영상 업로드
| 항목 | 내용 |
|------|------|
| 기능명 | 동영상 업로드 |
| 설명 | 사용자는 기술 데모나 튜토리얼 동영상을 업로드하여 게시물에 첨부할 수 있다. |
| 입력 | 동영상 파일 (mp4 등) |
| 처리 | Firebase Storage에 업로드, 인라인 플레이어로 재생 |
| 출력 | 게시물 내 동영상 플레이어 표시 |
| 예외 | 파일 형식 미지원 → 안내 메시지, 파일 크기 초과 → 안내 메시지 |

### F3-5. 해시태그
| 항목 | 내용 |
|------|------|
| 기능명 | 해시태그 |
| 설명 | 사용자는 게시물에 `#AI`, `#Python` 등의 해시태그를 붙여 주제를 분류할 수 있다. |
| 입력 | `#` 접두사가 붙은 키워드 |
| 처리 | 해시태그를 파싱하여 Firestore에 인덱싱, 클릭 시 해당 태그의 게시물 검색 |
| 출력 | 해시태그가 링크로 변환되어 클릭 가능 |
| 예외 | 특수문자 포함 시 해시태그로 인식하지 않음 |

## 화면 구성

| 화면 | 주요 요소 |
|------|-----------|
| 게시물 작성 모달 | 텍스트 입력 영역, 이미지/동영상 첨부 버튼, 해시태그 입력, 마크다운 도구 모음, 게시 버튼 |
| 게시물 상세 | 마크다운 렌더링된 본문, 코드 하이라이팅, 미디어 뷰어, 링크 프리뷰 카드 |
