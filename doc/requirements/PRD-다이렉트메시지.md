# PRD: 다이렉트 메시지 (DM)

## 기능 개요
사용자 간 1:1 또는 소규모 그룹 비공개 대화를 할 수 있는 메시징 기능입니다.

## 우선순위: 🟡 중요

## 상세 기능 명세

### F7-1. 1:1 다이렉트 메시지
| 항목 | 내용 |
|------|------|
| 기능명 | 1:1 DM |
| 설명 | 사용자는 다른 사용자에게 비공개 메시지를 보낼 수 있다. |
| 입력 | 메시지 텍스트, (선택) 이미지/코드 블록 첨부 |
| 처리 | Firestore 실시간 리스너를 통해 메시지 송수신 |
| 출력 | 채팅 인터페이스에 메시지 표시, 실시간 업데이트 |
| 예외 | 차단된 사용자에게 메시지 전송 불가 |

### F7-2. 그룹 DM
| 항목 | 내용 |
|------|------|
| 기능명 | 그룹 DM |
| 설명 | 사용자는 여러 사용자를 초대하여 소규모 비공개 그룹 대화를 할 수 있다. |
| 입력 | 참여자 선택, 메시지 텍스트 |
| 처리 | Firestore에 그룹 DM 채팅방 문서 생성, 참여자 전원에게 실시간 메시지 전달 |
| 출력 | 그룹 채팅 인터페이스 |
| 예외 | 참여자 수 제한 (최대 20명 등), 참여자 초대/퇴장 시 안내 메시지 |

### F7-3. 메시지 관리
| 항목 | 내용 |
|------|------|
| 기능명 | 메시지 관리 |
| 설명 | 사용자는 대화 목록 확인, 읽음 표시, 대화방 삭제 등을 할 수 있다. |
| 입력 | 대화 목록 탭 클릭, 개별 대화방 관리 메뉴 |
| 처리 | 읽음 상태 업데이트, 대화방 아카이브/삭제 |
| 출력 | 대화 목록에 읽지 않은 메시지 뱃지 표시, 마지막 메시지 미리보기 |
| 예외 | 삭제된 대화방은 상대방에게는 유지됨 |

## 화면 구성

| 화면 | 주요 요소 |
|------|-----------|
| DM 목록 | 대화 상대 프로필, 마지막 메시지 미리보기, 읽지 않은 메시지 뱃지, 시간 |
| 채팅 화면 | 메시지 버블 (상대/내 메시지 구분), 텍스트 입력 폼, 첨부 버튼 |
| 그룹 DM 생성 | 참여자 검색/선택, 그룹명 설정 |
