# PRD: 팔로우 및 소셜 관계

## 기능 개요
사용자가 다른 사용자를 팔로우하여 관심 있는 기술 콘텐츠를 지속적으로 받아볼 수 있는 소셜 관계 기능입니다.

## 우선순위: 🔴 필수

## 상세 기능 명세

### F2-1. 팔로우/언팔로우
| 항목 | 내용 |
|------|------|
| 기능명 | 팔로우/언팔로우 |
| 설명 | 사용자는 다른 사용자의 프로필이나 게시물에서 팔로우/언팔로우 버튼을 눌러 관계를 설정할 수 있다. |
| 입력 | 팔로우 버튼 클릭 |
| 처리 | Firestore에 팔로우 관계 문서 생성/삭제, 상대방에게 알림 발송 |
| 출력 | 버튼 상태 변경 (팔로우 ↔ 팔로잉), 팔로워/팔로잉 수 갱신 |
| 예외 | 자기 자신 팔로우 불가, 차단된 사용자 팔로우 불가 |

### F2-2. 팔로워/팔로잉 목록 조회
| 항목 | 내용 |
|------|------|
| 기능명 | 팔로워/팔로잉 목록 |
| 설명 | 사용자는 자신 및 다른 사용자의 팔로워/팔로잉 목록을 확인할 수 있다. |
| 입력 | 프로필 페이지에서 팔로워/팔로잉 수 클릭 |
| 처리 | Firestore에서 팔로우 관계 문서 조회 (페이지네이션) |
| 출력 | 사용자 목록 (프로필 사진, 닉네임, 자기소개, 팔로우 버튼) |
| 예외 | 비공개 계정의 경우 승인 후 열람 가능 (향후 확장) |

## 화면 구성

| 화면 | 주요 요소 |
|------|-----------|
| 팔로우 버튼 | 프로필, 게시물, 검색 결과 등에 팔로우/팔로잉 토글 버튼 표시 |
| 팔로워/팔로잉 목록 | 스크롤 가능한 사용자 리스트, 각 항목에 팔로우 버튼 포함 |
