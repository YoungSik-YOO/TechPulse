---
description: TechPulse 전체 기능 확인 및 테스트 프로세스 (E2E 및 수동 검증)
---

이 워크플로우는 구현 완료된 TechPulse SNS의 모든 기능을 시나리오별로 테스트하여 정상 동작 여부를 최종 확인합니다.

### 사전 조건
- 로컬 개발 서버(`npm run dev`) 또는 배포된 서버가 실행 중이어야 합니다.
- 테스트용 계정 2개 이상이 필요합니다.

---

## 1단계: 인증 및 계정 테스트

### 1-1. 회원가입 및 로그인 (SCR-001, 002)
- [ ] 새 이메일로 가입이 성공하는가?
- [ ] 중복 이메일 가입 시 에러 메시지가 표시되는가?
- [ ] 로그인 후 피드로 정상 이동하는가?
- [ ] 소셜 로그인(Google/GitHub)이 정상 작동하는가?

### 1-2. 프로필 설정 및 조회 (SCR-003, 004)
- [ ] 프로필 이미지 업로드가 성공하고 즉시 반영되는가?
- [ ] 닉네임, 소개 등 정보 수정이 Firestore에 저장되는가?
- [ ] 다른 사용자의 프로필 조회 시 편집 버튼이 숨겨지는가?

---

## 2단계: 핵심 기능(포스팅) 테스트

### 2-1. 게시물 작성 및 피드 (SCR-005, 006)
- [ ] 텍스트 게시물 작성이 성공하며 피드 상단에 나타나는가?
- [ ] **마크다운** 형식이 올바르게 렌더링되는가? (제목, 리스트 등)
- [ ] **코드 블록**에 하이라이팅이 적용되는가?
- [ ] 이미지/동영상 첨부가 정상적으로 표시되는가?
- [ ] 무한 스크롤이 작동하여 이전 게시물을 불러오는가?

### 2-2. 게시물 상세 및 댓글 (SCR-007)
- [ ] 게시물 클릭 시 상세 페이지로 이동하며 모든 내용이 보이는가?
- [ ] 댓글 작성이 성공하고 실시간으로 리스트에 추가되는가?
- [ ] 본인이 작성한 댓글만 삭제 버튼이 활성화되는가?

---

## 3단계: 그룹 및 커뮤니티 테스트

### 3-1. 그룹 탐색 및 생성 (SCR-008, 010)
- [ ] 새로운 그룹 생성이 성공하며 그룹 목록에 나타나는가?
- [ ] 카테고리/지역 필터링이 올바르게 작동하는가?

### 3-2. 그룹 상세 및 멤버십 (SCR-009)
- [ ] 그룹 가입/탈퇴가 정상적으로 처리되는가?
- [ ] 가입한 그룹원만 그룹 전용 게시물을 작성할 수 있는가?
- [ ] 비공개 그룹의 콘텐츠가 비멤버에게 노출되지 않는가?

---

## 4단계: 비기능적 요구사항 확인

### 4-1. 반응형 레이아웃
- [ ] 모바일 기기(또는 브라우저 검사기)에서 하단 네비게이션이 정상 표시되는가?
- [ ] 데스크톱에서 3컬럼 레이아웃이 깨짐 없이 표시되는가?

### 4-2. 보안 및 권한
- [ ] 로그아웃 상태에서 `/profile` 등 보호된 경로 접근 시 로그인 페이지로 튕기는가?
- [ ] 타인의 게시물을 수정/삭제하려고 시도할 때 차단되는가 (Firestore Rules)?

---
